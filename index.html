Samudra Saathi Platform Details

PS ID

SIH25039

Title

Samudra Saathi: A Crowdsourced Coastal Hazard Reporting & Analysis Platform

Description

  • A platform enabling citizens to report coastal hazards (like high waves, pollution, unusual activity) in real-time using a mobile/web app with photos, videos, and geotags.
  • It automatically monitors social media platforms for keywords and locations related to coastal incidents to capture public sentiment and early warnings.
  • An NLP & Analytics Engine processes all submitted data to classify reports, analyze sentiment, and automatically identify emerging "hotspots" of activity.
  • Provides a unified web dashboard for officials and analysts to visualize all data sources (citizen reports, social media trends, and official satellite data) on a single map.
  • Includes a proactive notification service that sends alerts (Push, SMS, Email) to relevant authorities when critical events are detected.

Data Sources Needed

  • Citizen Reports: Geotagged reports with media submitted via the app.
  • Social Media: Public posts from platforms like Twitter, filtered by keywords and location.
  • Government APIs:
    • ISRO/MOSDAC: For programmatic access to INSAT satellite imagery (e.g., cloud cover, sea surface temperature, wind vectors) for verification and context.
    • IMD (India Meteorological Department): For official data on weather forecasts, cyclone warnings, and rainfall.

Info to Gather from Government Organisations

  • Procedure and credentials for gaining API access to the MOSDAC (Meteorological and Oceanographic Satellite Data Archival Centre) portal to programmatically fetch INSAT satellite data products.
  • Documentation and access keys for IMD's public APIs to integrate official forecasts and warnings.
  • Information on using ISRO's Bhuvan geoplatform APIs for base maps and other relevant spatial data layers.

Possible Tech Stack

Kotlin Mobile App + MERN stack (MongoDB, Express.js, React.js, Node.js) configured Projects.
